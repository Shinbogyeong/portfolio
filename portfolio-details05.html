네트워크 구성도 설계

Project information

  • Category: Network Design
  • Project date: 21 Nov, 2018
  • Target: Company using DMZ

Project description

  • 프로젝트명
    회사 네트워크망 구성도 설계
  • 목적
    본사와 자사가 분리 되어있는 회사의 네트워크망을 설계한다.
  • 내용
    회사의 구성도를 고려해 각 부서별로 IP와 VLAN을 할당하고 회사 특성을 기반으로 네트워크망을 설계한다.
  • 차이점
    자사가 분리되어 있는 회사의 보안을 고려해 내부 네트워크와 분리된 서브 네트워크에서 공용 서비스를 노출하는 DMZ를 설치했다. Eather Channel을 통해 인터넷 서비스를 통해 발생하는 트래픽을 두 대의 백본에서 처리할 수 있도록 구성하여 로드밸런싱 효과를 얻을 수 있게 구성하였다. Server Farm으로 서버들을 클라이언트 망과 분리하여 따로 관리하고 허가받지 않은 외부 접근을 막고 안정화시켰다.

SIST 전산 관리 자바 콘솔 프로젝트

Project information

  • Category: Java Console Project
  • Language: JAVA (JDK 1.8)
  • Editor: Eclipse (Mars.2.Release(4.5.2))
  • Developer: 2명
  • Project date: 2021.05.01 - 2021.05.11

Project description

  • 프로젝트명
    SIST 전산 관리 자바 콘솔 프로젝트
  • 목적
    회사 직원들의 인적 사항, 일정, 급여, 연차, 게시판, 스케줄, 근태관리 등을 관리하기 위한 프로그램이다.
  • 차이점
    기존 전산 관리 시스템에 게시판, 스케줄 캘린더, 실시간 연차접수, 코로나 건강관리, 연말정산 등 다양한 기능을 추가해 차별성을 높였다.
  • 담당업무
    직원스케줄 캘린더 수정(관리자), 연차신청 결재 및 반려(관리자), 연차신청(직원), 스케줄 확인(직원)
  • 사용기술[Random, Math 클래스]
    데이터 생성을 위해 Random, Math 클래스를 사용하였다.
  • 사용기술[Calendar]
    직원 스케줄 캘린더를 출력하기 위해 Calendar.DAY_OF_WEEK를 사용해 구하고자 하는 달의 시작일의 요일을 구했다.
  • 사용기술[파일 입출력]
    직원이 연차신청을 할때 BufferedReader로 직원정보를 읽어와 해당 날짜에 사용 가능한 연차일수가 실시간으로 출력한다.
  • 사용기술[컬렉션]
    ArrayList T 을 활용하여 더미데이터를 쉽게 활용할 수 있도록 했고, ArrayList.add / ArrayList.remove 로 데이터를 새로 추가하거나 삭제할 수 있도록 하였다.
  • 느낀점
    처음 설계 오류로 더미를 여러 번 고쳤었는데 더미의 양이 많아서 데이터를 관리하기 힘들었다. 간단한 데이터 조작으로 출력문을 완성도 있게 고칠 수 있다는게 코딩할때 더 재밌는 요소로 다가왔다. 또 예외 처리(Exception Handling)로 프로그램 오류를 찾아보고 고민해보고 수정하고 해결했을 때 짜릿함을 잊을 수 없다.

SIST 전산 관리 자바 콘솔 프로젝트

Project information

  • Category: Java Console Project
  • Language: JAVA (JDK 1.8)
  • Editor: Eclipse (Mars.2.Release(4.5.2))
  • Developer: 2명
  • Project date: 2021.05.01 - 2021.05.11

Project description

  • 프로젝트명
    SIST 전산 관리 자바 콘솔 프로젝트
  • 목적
    회사 직원들의 인적 사항, 일정, 급여, 연차, 게시판, 스케줄, 근태관리 등을 더 편리한 환경에서 관리할 수 있는 프로그램이다.
  • 차이점
    기존 전산 관리 시스템에 게시판, 스케줄 캘린더, 실시간 연차접수, 코로나 건강관리, 연말정산 등 다양한 기능을 추가해 차별성을 높였다.
  • 담당업무
    직원스케줄 캘린더 수정(관리자), 연차신청 결재 및 반려(관리자), 연차신청(직원), 스케줄 확인(직원)
기술분야 상세내용
파일입출력 BufferedReader, Writer, Scanner
직원이 연차신청을 할때 BufferedReader로 직원정보를 읽어와 해당 날짜에 사용 가능한 연차일수를 실시간으로 출력한다
컬렉션 ArrayList<T>, HashMap<T>
ArrayList T 을 활용하여 더미데이터를 쉽게 활용할 수 있도록 했고, ArrayList.add / ArrayList.remove 로 데이터를 새로 추가하거나 삭제할 수 있도록 하였다.
Random, Math 클래스 데이터 생성을 위해 Random, Math 클래스를 사용하였다.
Calendar 직원 스케줄 캘린더를 출력하기 위해 Calendar.DAY_OF_WEEK를 사용해 구하고자 하는 달의 시작일의 요일을 구했다.
느낀점
프로젝트를 진행하면서 초반 데이터 설계가 중요한지 알 수 있었습니다. 필요한 데이터를 상세히 설정할수록 후반작업에서 데이터를 다양하게 활용할 수 있고, 세밀한 기능들을 화면에 출력할 수 있기 때문입니다. 또 각자의 스타일대로 만든 패키지명, 클래스명, 변수명은 혼란을 야기시켰습니다. 프로젝트 시작전에 서로 통일된 이름을 사용하기로 의논하는 것이 효율적인 프로젝트로 만들수 있다는 것을 배우게 되었습니다.

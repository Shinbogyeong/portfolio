Project information

  • Category: Web design
  • Language: HTML, JavaScript, Css, Oracle
  • Editor: Atom
  • Developer: 2명
  • Project date: 05 June, 2017

Project description

  • 프로젝트명
    건강관리진단과 스포츠 강의 수강 웹 서비스
  • 목적
    다양하게 개설된 스포츠 강의에 대한 설명과 편리한 수강신청을 통해 보다 쉽게 건강관리를 할 수 있다.
  • 차이점
    기존의 헬스장 홈페이지와 달리 비만도를 측정할 수 있어 손쉽게 자신의 건강 상태를 체크할 수 있으며 각 비만도에 따라 다양한 관리법을 추천한다. 또한 DB와 연동해 실시간 수강신청 인원을 출력한다.
  • 담당업무
    HTML과 CSS를 이용한 웹 화면구현
  • 사용기술(수강신청)
    a:hover를 이용해 각 운동 아이콘에 커서를 올리면 강사의 약력이 나오게 구성하였고 하단부에 DB와 연계해서 현재 수강신청 인원을 나타냈다.
  • 사용기술(강좌소개)
    a 태그의 href 속성으로 강의의 커리큘럼이 적힌 사이트로 이동시킨다.
  • 사용기술(비만도 측정)
    성별에 따라 비만도를 구분하였고 input 태그의 type 속성을 range로 설정해 본인의 비만도 수준을 살필 수 있다.

학사 관리프로그램

Project information

  • Category: Oracle
  • laguage: SQL, JDBC
  • Editor: SQLdeveloper-20.4.1
  • Developer: 5명
  • Project date:2021.05.24 - 2021.06.04

    Project description

  • 프로젝트명
    학사 관리프로그램
  • 목적
    학생, 교사, 관리자가 사용하는 교육센터에서 프로그램이다.
  • 차이점
    교육 종료 교육생의 취업현황을 관리하는 테이블을 만들어 교육생이 취업관련 자료를 쉽게 열람할 수 있다.
  • 담당업무
    ERD설계, DDL(CREATE, DROP, SELECT) 작성, DML(Trigger), Procedure , Function, View) 작성
기술분야 상세내용
Function 더미 데이터에서 주민번호를 읽어와 substr으로 남자와 여자를 구별해 데이터를 출력하는 함수를 만들었다.
Procedure tblEnrollment, tblStudent, tblOpenCourse를 inner join을 통해 레코드를 결합시켜
과정이 끝난 교육생을 tblCompleteStudent에 추가하는 프로시저를 만들었다.
View 개설과정테이블(tblOpenCourse)의 endate 속성에서 레코드 값과
to_char(sysdate, 'yy-mm-dd')라는 문자 형식으로 변환 된 날짜 함수와 비교해서 크기별로 다른 값을 출력한다.
느낀점
실제 교육센터 이용경험을 바탕으로 관리자, 교사, 교육생들에게 필요한 기본적인 기능들과 추가적으로 교육종료 교육생들에게 필요한 채용지원 현황을 구현했습니다.
처음 진행해보는 데이터베이스 프로젝트이기 때문에 ERD모델링에 가장 오랜 시간을 투자했습니다. 이론에서 배웠던것을 바탕으로 테이블끼리 어떻게 참조해야 할지 어떤 연결을 주어야 할지 부모-자식관계는 어떻게 형성되는지를 고려해서 설계를 진행했습니다.
업무를 진행하다 보니 테이블에 컬럼들을 수정하거나 check절에 제약사항을 추가가 필요해 DDL문을 변경해야 하는 문제점이 발생하기도 했지만, CODE TOGETHER을 설치해 팀원들과 코드를 공유하면서 쿼리문을 효율적으로 수정하고 추가할 수 있었습니다.

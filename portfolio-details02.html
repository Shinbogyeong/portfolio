학사 관리프로그램

Project information

  • Category: Oracle
  • laguage: SQL, JDBC
  • Editor: SQLdeveloper-20.4.1
  • Developer: 5명
  • Project date:2021.05.24 - 2021.06.04

    Project description

  • 프로젝트명
    학사 관리프로그램
  • 목적
    학생, 교사, 관리자가 사용하는 교육센터에서 프로그램이다.
  • 차이점
    교육 종료 교육생의 취업현황을 관리하는 테이블을 만들어 교육생이 취업관련 자료를 쉽게 열람할 수 있다.
  • 담당업무
    ERD설계, DDL(CREATE, DROP, SELECT) 작성, DML(Trigger), Procedure , Function, View) 작성
  • 사용기술[Function]
    더미 데이터에서 주민번호를 읽어와 substr으로 남자와 여자를 구별해 주는 함수를 만들었다.
  • 사용기술[Procedure]
    tblEnrollment, tblStudent, tblOpenCourse를 inner join을 통해 레코드를 결합시켜 과정이 끝난 교육생을 tblCompleteStudent에 추가하는 프로시저를 만들었다.
  • 사용기술[View]
    개설과정테이블(tblOpenCourse)의 endate 속성에서 레코드 값과 to_char(sysdate, 'yy-mm-dd')라는 문자 형식으로 변환 된 날짜 함수와 비교해서 크기별로 다른 값을 출력한다.
  • 느낀점
    프로젝트 과정 중에서 ERD모델링에 가장 오랜 시간을 투자했다. 테이블끼리 어떻게 참조해야 할지 어떤 연결을 주어야 할지 고민해서 설계하면 다음 날 또 맹점을 발견해 처음부터 다시 설계하기를 여러 번 했다. 다 만들었을때도 아쉬운 점이 있어서 탄탄한 뼈대의 중요성을 깨달았다.
